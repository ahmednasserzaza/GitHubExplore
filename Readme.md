# GitHubExplore

an app that shows public repositories on github

## Technology Used

- Kotlin
- Jetpack Compose
- MVVM architecture pattern

## Libraries

- Kotlin Coroutines
- Dagger Hilt dependency injection
- Retrofit
- Coil
- Lottie
- Kotlin LocalDateTime
- Shimmer effect
